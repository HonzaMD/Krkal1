Instalace

Emulator DOSBox 0.63+ (http://dosbox.sourceforge.net/) Doporucujeme!
  • Nainstalujte si emulator, seznamte se s nim, spravne ho nataktujte (mne to fungovalo na 7500 cyklech).
  • Stahnete a rozbalte Krkala.
  • Pod emulatoram nastavte parametry sve zvukove karty: Spustte msetup.exe (zvolte SoundBlaster 16, dalsi veci nechte tak, jak jsou, zvolte stereo 16 bit)
  • Spustte krkal.exe a Hrajte!
Windows
  • Spustte msetup.exe a zvolte nosound! (Je nutno vypnout hudbu, jinak to spadne!)
  • Ujistete se ze mate zapnutou podporu expanded memory (EMS)
  • Spustte krkal.exe a Hrajte!
DOS, Stary pocitac 486, ISA zvukova karta
  • spravne si nakonfigurujte vas DOS (nutny je hlavne ovladac expanded memory (EMS), mys, zvukova karta)
  • Spustte msetup a spravne nastavte parametry sve zvukove karty
  • Spustte krkal.exe a Hrajte!
